# 📖 Ruby Morse Decoder <a name="about-project"></a>

This project is a Ruby program that demonstrates the use of modules and classes to implement common Enumerable methods such as #all?, #any?, and #filter. It also implements a custom #each method in the MyList class that uses the MyEnumerable module. The program takes input in the form of a list of integers and outputs the results of calling the implemented methods on that list.
